```python
#LSTM模型参数
torch.nn.lstm(input_size,hidden_size,num_layers,bias,batch_first,dropout,bidirectional)
input_size: 输入数据的特征维数，通常就是embedding_dim(词向量的维度)
hidden_size: LSTM中隐层的维度
num_layers: 循环神经网络的层数
bias: 用不用偏置，default=True
batch_first: 默认是False,输入需要将batch_size与seq_length这两个维度调换，方便并行计算
dropout: 默认是0，代表不用dropout
bidirectional: 默认是false，代表不用双向LSTM
#输入
input(seq_len, batch, input_size)
h0(num_layers * num_directions, batch, hidden_size)
c0(num_layers * num_directions, batch, hidden_size)
#输出
output(seq_len, batch, hidden_size * num_directions)
hn(num_layers * num_directions, batch, hidden_size)
cn(num_layers * num_directions, batch, hidden_size)
```

> **Bert模型介绍** 
> - bert-base-chinese：中文   
>   bert-base-cased：支持大小写   
>   bert-base-uncased：仅支持小写（需要进行lower处理） 
> - 配置文件 —— config.json BertConfig
>   模型文件 —— *.bin BertModel  
>   词表文件 —— vocab.txt bert名称加载（运行时从官方数据库下载）
> - 本地的bert文件夹地址（本地的bert文件夹下的config等文件地址）  
>   在预训练过程中，线性层权值会根据你的任务进行参数更新 
>   (32000+512+2)x768+(3x768x64x12+768x768+768x4x768x2)=110M

```python
#BERT模型调用方式
from transformers import BertTokenizer, BertModel,BertConfig,BertForSequenceClassification
tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')# 加载分词表->设计数据集
model = BertModel.from_pretrained("bert-base-uncased")# 加载bert预训练模型
config = BertConfig.from_pretrained('bert-base-chinese')# 加载模型的参数->修改模型的默认参数
classModel = BertForSequenceClassification.from_pretrained('bert-base-chinese',)# 加载下游的分类模型
#参数
forward(input_ids=None, attention_mask=None, token_type_ids=None, position_ids=None, head_mask=None, 
inputs_embeds=None, encoder_hidden_states=None, encoder_attention_mask=None, past_key_values=None, use_cache=None,
 output_attentions=None, output_hidden_states=None, return_dict=None)
# 输入
input_ids: 必须有,torch.LongTensor(batch_size, sequence_length)输入文本在词典中的映射id，又称tokens
attention_mask：torch.FloatTensor(batch_size, sequence_length)避免对padding(填充，因为输入需要时定长，不够长度的，用0补齐)的tokens索引过于关注
token_type_ids：torch.LongTensor(batch_size, sequence_length)用来区分一条样本中第一部分和第二部分
position_ids：torch.LongTensor (batch_size, sequence_length)在位置embedding中，每个输入序列tokens的位置索引
# 输出
return_dict=True,则返回一个BaseModelOutputWithPoolingAndCrossAttentions 
last_hidden_state：torch.FloatTensor (batch_size, sequence_length, hidden_size)模型最后一层的hidden_state
pooler_output：torch.FloatTensor(batch_size, hidden_size)最后一层由线性层和Tanh激活函数进一步处理过的序列的第一个token(分类token)的隐藏状态
```
